Welcome!

Hey hey hey! Welcome to the offline editable release of the tutorial. You can study it on Cook islands being completely out of the net, and everything would work. To start browsing, go to the language folder and open nav.html.

TroubleShooting

For Chrome/Chromium: run the browser with the flag --allow-file-access-from-files. For other issues and suggestions, use Github at https://github.com/iliakan/javascript-book/.

Contribution

Help us to create a really great JavaScript book!

You are welcome to:
  • Fix typos.
  • Translate the book.
  • Rephrase and improve the text, make it better.
  • Add your chapters.

The contributions are carefully reviewed and appended or integrated with the current content.

Sending changes

All changes are done offline. This way is chosen, because it gives full freedom.

You can add images, HTML/JS/CSS examples and manage the text in your favorite editor. The best way to send your changes is Git. If you know it, then use pull requests. If you don't know it... Things are still easy.
  1. First, changes in any form are accepted.
The easiest way to send changes is to use Git.

The best way is to Git and create pull request for the repo at https://github.com/iliakan/javascript-book/ , or if you are unfamiliar with Git, send your updates to iliakan@javascript.info.

By sending out your contribution, you agree to the terms of the CC0 license. In short, you provide no warranty of any kind and give it to the public domain. If you want to contribute a large part under another license - let us know.

Thank you.
